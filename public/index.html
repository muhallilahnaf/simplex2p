Two-phase simplex calculator

with detailed steps!

Enter the linear program in the input box

Use the following syntax:

First part: {min/max} = {objective function}

Example:
  • Specify minimization or maximization using "min" or "max"
  • Write the objective function expression
  • Keep both of them in a single line, and separate min/max and the expression by "=" sign

Second part: constraints

Example:
  • Write each constraint in a separate line
  • No need to write non-negativity constraints
  • No need to write "subject to" or similar phrases
  • Avoid using the following variable names: slck, srpls, artfcl as they are used in algorithm for slack, surplus and artificial variables respectively
